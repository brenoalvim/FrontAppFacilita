### BACK-END (MySql ou PHPMyAdmin, PHP, Laravel, xampp)

Requisitos:

PHP 7.4

composer 2.2.14

xampp 7.4.29-1

~~~
Start Apache
~~~

~~~
Start MySQL
~~~

~~~
Criar banco de dados com o nome "library"
~~~

~~~
git clone https://github.com/brenoalvim/api_app_facilita
~~~

~~~
cd api_app_facilita
~~~

~~~
composer install 
~~~

~~~
php artisan migrate
~~~

~~~
php artisan serve
~~~
